# Udemy Course Price Prediction 📚💰

A Machine Learning application that predicts **Udemy course prices** based on course features using a trained regression model. The project includes data preprocessing, exploratory data analysis, feature engineering, model training, and an interactive Streamlit web application for real-time predictions.

**Python • Streamlit • Scikit-learn • Pandas • NumPy • Matplotlib**

---

## 📖 Overview

This project predicts the prices of Udemy courses based on their characteristics. It uses Machine Learning techniques to analyze the dataset, train a regression model, and provide users with an easy-to-use interface for estimating course prices.

## 🚀 Quick Start

### Requirements

- Python 3.10+
- pip

### Clone the Repository

```bash
git clone https://github.com/YourUsername/Udemy-Course-Price-Prediction.git

cd Udemy-Course-Price-Prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
